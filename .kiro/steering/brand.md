# Brand & Design

## Tone of Voice: "Funny and Competent"

- Confident but not arrogant
- Witty but not try-hard — humor emerges from context, never forced
- Clear but not dry — concise with warmth
- Casual but not sloppy — informal-professional register

### Do
- Be direct and clear
- Use humor that emerges naturally from context
- Show personality through choices, not declarations
- Let technical depth speak for itself

### Don't
- Use corporate jargon or buzzwords
- Force jokes or use memes
- Say "I'm creative" or "I'm passionate"
- Break the fourth wall constantly
- Over-explain or talk down

## Visual Identity: Pixel Art + Modern UI

- **Pixel art**: World, characters, locations, sprites, decorative elements. Brand personality.
- **Modern UI**: Content presentation, navigation, cards, typography, functional interfaces. Professionalism.
- Classic view = modern UI with pixel-art accents
- Play view = pixel art with modern UI overlays (dialogue, panels, menus)
- Single palette, no dark/light theme toggle

## UX Principles (Quick Reference)

1. Clarity first
2. No gates, only invitations
3. Dual navigation, shared destination
4. Conversion always accessible
5. Exploration rewards curiosity
6. Progressive disclosure
7. Frictionless content access
8. Accessibility as baseline

## UI Principles (Quick Reference)

1. Two visual languages, one identity
2. Content readability above all
3. Density without clutter
4. Consistent component language
5. Visual hierarchy serves the goal
6. Cross-view coherence
7. Responsive as requirement
8. Motion with purpose
9. Pixel art quality standards
10. Single palette, no theme toggle
